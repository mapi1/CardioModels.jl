# CardioModels

## Notation
The following convention is used for cardiovascular variables:

* I:  RR Intrval series
* S:  Systolic blood pressure
* D:  Diastolic blood pressure
* P:  Pulse Pressure
* $\rho$:   Respiration
* T: Time constant as a measure of peripheral resistance R
